# 质量数据分析看板

这是一个前端静态数据分析页面，用于对质检 Excel/CSV 数据做可视化分析与筛选。

## 功能概览

- 上传 `.xlsx` / `.xls` / `.csv` 文件
- 自动合并 `检测项名称 + 不良名称` 为 `合并项`
- 顶部筛选条联动分析（时间范围、车型、供应商、状态）
- 车型问题项目数饼图
- TOP5 供应商明细（按项目数统计）
- 不良原因帕累托图（堆积柱 + 累计占比折线）
- 点击帕累托图柱体查看问题点明细
- 表格自动合计行
- 导出当前筛选下统计结果

## 目录说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `script.js`：上传、清洗、筛选、图表与导出逻辑
- `vercel.json`：部署配置

## 本地打开

这是纯前端页面，无需安装依赖。可直接双击 `index.html` 打开，或在任意静态服务器中打开。

## 数据字段建议

建议源数据包含以下列（部分支持自动识别）：

- `检测项名称`
- `不良名称`
- `车型`
- `供应商`
- `不良等级`
- `解决进度`
- `发生原因`（可选）
- `不良状态`（可选）
- `录入时间/日期`（可选，用于时间筛选）
